    v1.2.5 execution backlog — resiliency, ETL stability, no-code rules¶

    Branch train: ide/v1.2.5-7320 (merge epics via focused PRs; link each PR to this doc section).

    Ordering (fastest ROI first): ETL contract + quality gates → no-code rule workflow → degradation/SLO burn → rule telemetry → chaos + replay certification.

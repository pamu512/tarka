**Medium tips**

- Title and subtitle: set in Medium’s editor, not as `#` in the body.
- After paste, select each `##` line and apply **Heading (large)** twice for H2, or **Heading** once depending on theme.
- Code blocks: ensure triple backticks survived; Medium sometimes needs you to re-apply **Code** style.
- Diagrams: ASCII art is monospace; highlight the block and use **Code** or a fixed-width paragraph. For crisper graphics, screenshot the block or use `docs/docs/assets/tarka-evaluate-architecture.svg`.

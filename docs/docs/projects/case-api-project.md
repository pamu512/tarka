# Case API Project

## Scope

Case lifecycle, dispute operations, queueing, audit trails, and analyst workflow APIs.

## Current Gaps

- Outcome feedback loops into model governance are still shallow.
- Bulk workflow ergonomics and SLA intelligence can be improved.

## Roadmap

### Now

- Strengthen dispute outcome capture quality and consistency.
- Improve case audit payload quality for explainability workflows.

### Next

- Add richer queue score decomposition and recommended-action confidence.
- Add more robust workflow simulation and rollback safety.

### Later

- Adaptive case routing using cross-feature risk context and workload balancing.
